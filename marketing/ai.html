Capabilities

What gets generated

⚡

API Client

Axios instance with base URL, timeout, auth interceptors, refresh token, retry logic, and typed error mapping.

🔧

Service Layer

Full CRUD classes per resource — getAll, getById, create, update, delete — with query params, path params, multipart uploads, file downloads, pagination, sorting, and filtering.

⚛️

React Query Hooks

useUsers, useUser, useCreateUser, useUpdateUser, useDeleteUser — with infinite queries, optimistic updates, prefetch helpers, cache config, and invalidation.

📐

TypeScript Types

Request types, response types, enums, generic API response wrappers, pagination models, error models. Strict mode. Zero `any`.

✅

Zod Validators

Request payload schemas, form validation schemas with React Hook Form integration, optional response validation.

🎭

MSW Mocks

Service Worker handlers with Faker-generated realistic data. Develop frontend without any backend availability.

🔐

Auth Detection

Auto-detects JWT, Bearer, Cookie, OAuth, API Key. Generates login/logout services, refresh logic, auth middleware, and protected request helpers.

🧪

Vitest Tests

Starter test suites for services, hooks, mock server, and the API client. Ready to extend with business logic.

🔄

API Diff Engine

Compare two specs — detect added/removed endpoints, schema changes, deprecations, and breaking changes. Generates a migration report.

CLI

One command to start.

Every workflow has a dedicated command. Composable with CI/CD.

$ api-gen init scaffold config file
$ api-gen import swagger.json parse + generate everything
$ api-gen sync regenerate from saved spec
$ api-gen generate users generate single resource
$ api-gen diff old.json new.json detect breaking changes
Configuration

Fully configurable.

Drop api-generator.config.ts in your project root. Every option has a sensible default.

api-generator.config.ts
import { defineConfig } from 'api-gen' export default defineConfig({ input: './swagger.json', output: './src', client: 'axios', // 'axios' | 'fetch' validation: 'zod', // 'zod' | 'yup' queryLibrary: 'react-query', // '@tanstack/react-query' v5 testRunner: 'vitest', generate: { services: true, hooks: true, validators: true, mocks: true, tests: true, docs: true, }, naming: { convention: 'camelCase', // | 'snake_case' servicesSuffix: '.service', hookPrefix: 'use', }, })
